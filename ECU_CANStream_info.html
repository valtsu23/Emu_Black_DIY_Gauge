EMU CAN STREAM


EMU device is able to send data that can be used by external data loggers or Dashboards using CAN-BUS.
Default values of CAN IDs can be modified in options EMU CAN Data


CAN BUS speed

Selectable: 125kbps, 250kbps, 500kbps, 1Mbit

Message format

11 bit standard

Data format

INTEL (Little endian)
